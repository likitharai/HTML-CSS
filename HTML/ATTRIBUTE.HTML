Attribute Example

HTML attributes are used to provide additional information about elements.

This is an example of using attributes in HTML.

Attributes can define properties such as color, alignment, and more.

For example, the bgcolor   attribute sets the background color of the page.
Other common attributes include align  , width  , and height.
